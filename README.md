# Sentiment Liquidity Engine

Motor de trading para EUR/USD basado en:

- Liquidez (sweeps)
- Market Structure
- DXY correlation
- Confluencia de contexto
- Ranking de señales

## 🚀 Características

- Dashboard en Streamlit
- Alertas visuales y sonoras
- Guardado automático de señales
- Filtro de setups válidos
- Integración DXY

## ▶️ Cómo ejecutar

```bash
pip install -r requirements.txt
streamlit run app.py
